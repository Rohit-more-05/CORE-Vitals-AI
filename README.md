# ❤️ CORE-Vitals AI: Heart Stroke Prediction

An advanced, cyberpunk-styled Streamlit web application that uses a K-Nearest Neighbors (KNN) Machine Learning model to evaluate a patient's risk of suffering a heart stroke based on clinical and lifestyle parameters.

![UI Preview](https://img.shields.io/badge/UI-Cyberpunk_Bento_Grid-00f2ff?style=for-the-badge)
![Python](https://img.shields.io/badge/Python-3.8%2B-blue?style=for-the-badge&logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.x-FF4B4B?style=for-the-badge&logo=streamlit)

## 🌟 Features
- **Machine Learning Inference:** Employs a trained KNN model to accurately classify heart stroke risk.
- **Cyberpunk Aesthetics:** A sleek, premium dark-mode interface with glassmorphism bento grids and neon accents.
- **Interactive Visualizations:** Uses `plotly` to render an animated, color-coded risk gauge chart.
- **Categorized Inputs:** Patient parameters are intuitively split into Vitals, Clinical Tests, and Lifestyle tabs.

## ⚙️ Local Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rohit-more-05/heart-stroke-prediction.git
   cd heart-stroke-prediction
   ```

2. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   streamlit run app.py
   ```

## 🛠️ Built With
- **Python** & **Pandas** for data manipulation
- **Scikit-Learn** (via `joblib`) for the pre-calculated KNN model and scaler
- **Streamlit** for the frontend application framework
- **Plotly** for high-end gauge visualisations
